# Nanoservices Diploma Project

## Getting started

App is available at: <http://smarthome-front.s3-website.eu-central-1.amazonaws.com/>

## Build packages

````sh
cd ./src/backend
./build.ps1
````

## Deploy backend

````sh
cd ./src/backend
serverless deploy
````

## Deploy frontend

````sh
cd ./src/frontend
serverless deploy
````
